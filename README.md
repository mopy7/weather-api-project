# WHAT'S THE WEATHER

A minimalist, monochrome weather application inspired by dot-matrix aesthetics.

## FEATURES
- **REAL-TIME CLOCK**: 24-HOUR PRECISION_
- **MINIMALIST UI**: DOT-MATRIX BACKGROUND & MONOCHROME PALETTE_
- **SIMULATED WEATHER**: INSTANT PREVIEW OF VARIOUS CONDITIONS_
- **RESPONSIVE DESIGN**: ADAPTS TO MOBILE & DESKTOP_

## STACK
- **HTML5**: SEMANTIC STRUCTURE_
- **CSS3**: CUSTOM DOT-GRID SYSTEM & GLASSMORPHISM_
- **JAVASCRIPT**: ASYNC/AWAIT FETCH SIMULATION_

## SETUP
1. CLONE THE REPOSITORY_
2. OPEN `index.html` IN ANY MODERN BROWSER_
